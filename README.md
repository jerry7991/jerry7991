## Hard Problems I've Solved

🐢→⚡ **Device session startup: 63% faster**  
Traced a multi-system performance bottleneck across the streaming pipeline, device automation agent, iOS/Android instrumentation layers, and session pooling. Optimized across 5 subsystems end-to-end — root cause to production rollout.

🍎 **Eliminated a silent iOS 18+ crash affecting enterprise banking clients**  
Apple's OS update silently changed the Settings navigation hierarchy — enterprise apps were being incorrectly terminated. Built a recursive, depth-unlimited UI hierarchy scanner deployed cross-layer across 3 repos behind a feature flag.

🤖 **Built an AI agent that handles ~25% of customer support tickets autonomously**  
Grounded debug agent backed by a structured knowledge base (53 versions, decision trees) + 6 live data source integrations via MCP protocol. Self-improving: every resolved ticket strengthens the agent.

💾 **Fixed a deep memory leak — reduced memory-related incidents by ~86%**  
Traced and eliminated a systemic memory leak in a high-throughput Ruby on Rails service. Container crash rate dropped from >7× baseline to normal; app restarts reduced by 60%.

💰 **Cut infra cost by ~25% relative to 23% platform growth**  
Eliminated redundant telemetry pipelines and right-sized cloud resources (AWS + CDN). Net cost reduction even as platform traffic grew — cost efficiency improved ~40%.

🔕 **Reduced alert noise by 27% across a distributed microservices platform**  
Moved from reactive silencing to category-based threshold engineering. Permanently improved on-call quality across 6 services.

🌐 **Designed a transparent TCP proxy for proxy-unaware mobile apps**  
Used pfctl rules + Privoxy integration to intercept both proxy-aware and proxy-unaware iOS app traffic on macOS — without modifying app source code. Published open-source → [`transproxy`](https://github.com/jerry7991/transproxy)
